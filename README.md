# 🎓 Student Management System (Java)

A Java Swing-based desktop application for managing student records.

## ✨ Features
- Add / Edit / Delete students
- GPA calculation
- Grade management
- Search (Name & ID)
- Sorting (Name & GPA)
- CSV Export
- Data persistence using serialization
- Login system

## 🛠 Technologies Used
- Java
- Swing (GUI)
- File Handling (Serialization)

## 🚀 How to Run
cd src
javac SMS/*.java
java SMS.MainGUI
